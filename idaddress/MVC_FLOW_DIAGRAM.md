## 安全层级

```
┌─────────────────────────────────────────┐
│          最高安全级 - 私钥               │
│  - 绝对保密                              │
│  - 控制资金                              │
│  - 不可恢复                              │
└─────────────────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│        中等安全级 - 公钥/地址            │
│  - 可以公开                              │
│  - 仅用于接收                            │
│  - 无法控制资金                          │
└─────────────────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│        公开信息 - 交易记录               │
│  - 完全公开                              │
│  - 可在区块链查询                        │
│  - 无法修改历史                          │
└─────────────────────────────────────────┘
```

## 快速参考

### 常用命令

```bash
# 生成密钥
./mvckey generate

# 地址转换
./mvckey convert <地址>

# 恢复地址
./mvckey info <私钥hex>

# 完整测试
./test_flow.sh
```

### 文件结构

```
cmd/mvckey/
├── main.go           # 主程序
├── test_flow.sh      # 测试脚本
├── README.md         # 使用文档
├── .gitignore        # Git忽略规则
└── test_account_*.json  # 生成的账户(不提交)
```

### 关键路径

```
生成 → 导入 → 充值 → 转账 → 验证
 ↓      ↓      ↓      ↓      ↓
mvckey wallet  tx    wallet  explorer
```
