### 支持WEB图形界面
```
./metaid-cli web --chain btc --port 8080
./metaid-cli web --chain doge --port 8081
```

### 完整的命令：
   - `metaid-cli init` - 初始化
   - `metaid-cli address create/list` - 地址管理
   - `metaid-cli utxo add/list` - UTXO 管理
   - `metaid-cli inscribe create` - 创建铭文，交互创建
   - `metaid-cli broadcast` - 广播交易
   - `metaid-cli wallet show` - 查看钱包
